## ◈ &nbsp; What is this?

**Midnight Obsidian** is not a portfolio — it's a **narrative experience**.

Each scroll advances a cinematic story about who I am and what I build. The universe opens with a 3D void, then guides you through four distinct phases: the interfaces I've crafted, the intelligence I've architected, the data I've decoded, and finally — a direct terminal link into my core.

No generic cards. No cookie-cutter layouts. Just a single, immersive story told through code, motion, and light.


**Effects used:**
- `Glassmorphism` — translucent layers with `backdrop-filter: blur`
- `Noise Overlay` — SVG fractal grain texture for depth
- `Radial Gradients` — soft glow emanating from scene centers
- `Spring Physics` — `useSpring` for scroll-smoothing and momentum
- `Error Boundary` — Spline failures fall back to animated gradient, never crash

## ◈ &nbsp; Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/ShreyasVavley/profile.git
cd profile

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

Open **[http://localhost:5173](http://localhost:5173)** in your browser.

```bash
# Production build
npm run build

# Preview the production build locally
npm run preview
```
